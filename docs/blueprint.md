# **App Name**: Forge CRM

## Core Features:

- Lead & Account Management: Create, view, and update leads with their status, origin, AI score, and interaction history, linking them to accounts.
- Opportunity Pipeline (Kanban): Visualize and manage sales opportunities in a responsive Kanban board, tracking value and expected close dates.
- Activity Tracking: Log and review all interactions (calls, emails, meetings) associated with leads and opportunities.
- AI-Powered Opportunity Summaries: Automatically generate concise summaries of opportunity progress and details based on logged activity notes using a generative AI tool.
- Role-Based Access Control: Manage user access and permissions for admin, manager, and sales roles to ensure data security and proper workflow.
- Performance Dashboard: Display key sales metrics, including conversion rates and average sales cycle time, for insights into performance.
- Firebase Data Integration: Utilize Firebase Firestore for robust data storage and Firebase Auth for secure user management.

## Style Guidelines:

- Primary Blue (#023567) for core elements and primary actions, conveying professionalism and reliability.
- Primary Pink (#F8B8B9) as a secondary accent or for specific highlights, offering a modern contrast.
- Action Blue (#4A90E2) for interactive elements, buttons, and important notifications, ensuring visibility and user engagement.
- Dark Slate for text, dark mode backgrounds, or other deep neutral elements.
- Background color: A very light, cool grey (#EFF0F2), providing a clean and understated canvas that highlights content and ensures readability, particularly in light mode.
- Headlines and prominent text: 'Montserrat' (sans-serif), chosen for its strong presence and modern, geometric aesthetic.
- Body text: 'Inter' (sans-serif), for its excellent readability and neutral, professional appearance across various content types.
- Employ a minimalist and crisp icon set to maintain a clean visual design, ensuring clear communication of actions and statuses without clutter.
- Implement a responsive, adaptive layout that optimizes content presentation across desktop, tablet, and mobile devices, with particular attention to the Kanban view for opportunities and dashboard metrics.
- Subtle and functional animations for state changes, transitions, and user feedback, ensuring a smooth and intuitive user experience without distracting from the data.
